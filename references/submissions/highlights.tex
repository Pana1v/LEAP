\begin{itemize}
\item LEAP prunes solver arcs with a graph neural network for exact pick-and-place.
\item Arc pruning cuts solver variables from order N-squared to order N times k.
\item 8.75x faster than the exact solver at 200 objects, with 0.058% empirical gap.
\item A learned graph formulation captures asymmetric pick-place-feedback costs.
\item k=15 gives the best quality-speed balance across problem sizes in tests.
\end{itemize}
